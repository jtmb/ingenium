# Ingenium Documentation

Welcome to the Ingenium documentation. Ingenium is a self-learning AI agent skill system and MCP server with integrated email client support. It provides skills, observations, tasks, context, plugins, servers, and project management through a single MCP stdio transport, with a Next.js dashboard for visual management.
